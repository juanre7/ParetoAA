# Privacy Policy

Last updated: June 22, 2026

Artificial Analysis Pareto Overlay is a Chrome extension that adds a Pareto frontier overlay and comparison table to scatter charts on `https://artificialanalysis.ai/`.

## Data processed by the extension

The extension processes website content from Artificial Analysis pages locally in the user's browser. This includes visible chart elements such as SVG points, chart labels, axis text, model labels, and related page structure needed to calculate and display the Pareto frontier.

The extension also stores display preferences locally in the browser, such as whether the overlay is enabled, whether dominated points are dimmed, whether the table is shown, and how chart axes should be interpreted.

## Data not collected

The extension does not collect, transmit, sell, or share user data.

The extension does not collect:

- personally identifiable information
- health information
- financial or payment information
- authentication information
- personal communications
- location data
- web browsing history
- user activity across websites

## How data is used

Website content is used only to provide the extension's single purpose: identifying visible chart points on Artificial Analysis, calculating non-dominated Pareto points, and rendering the overlay and comparison table.

Display preferences are used only to remember the user's local extension settings.

## Data sharing and transfers

No user data is sent to the developer, to external servers, or to third parties.

The extension does not use user data for advertising, analytics, creditworthiness, lending, profiling, or any purpose unrelated to the extension's single purpose.

## Remote code

The extension does not use remote code. All executable code is included in the extension package.

## Contact

For privacy questions, contact the publisher through the contact email listed on the Chrome Web Store developer page for this extension.
